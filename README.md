# Emprestimo

Projeto para Modulo de POO  desenvolimento de Emprestimo usando NodeJS e Typescript


## Instalação

Instale my-project com npm

```bash
  npm install 
 
```
    
## Demonstração


Para executar o projeto 
```bash
  npm run start
 
```
## Stack utilizada

**Back-end:** Node, Typescrpit
